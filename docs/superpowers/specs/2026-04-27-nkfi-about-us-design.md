# NKFI About Us 페이지 디자인 사양서

## 1. 개요
- **목적:** NKHB의 모체인 NKFI(뉴코리아 파운데이션 인터내셔널)와 설립자 케네스 배 선교사에 대한 상세 정보를 제공하여 단체의 신뢰성과 사명을 알림.
- **대상 파일:** `about.html` (신규 생성)
- **주요 내용:** NKFI 소개, 비전/사명, 4대 사역, 설립자 프로필.

## 2. 디자인 전략
- **톤 앤 매너:** 신뢰감 있는 네이비와 화이트 톤, 정돈된 레이아웃, 감동적인 스토리텔링.
- **레이아웃:** 반응형 원페이지 스크롤 (섹션별 구성).

## 3. 페이지 상세 구조

### 3.1 Hero Section
- **배경:** 짙은 네이비 배경 또는 부드러운 화이트.
- **타이틀:** `NKFI (뉴코리아 파운데이션 인터내셔널)`
- **슬로건:** "북한 주민의 회복, 자유, 그리고 존엄을 향하여"
- **네비게이션:** 메인 페이지로 돌아가기 버튼 포함.

### 3.2 Introduction Section (NKFI 소개)
- **내용:** 2016년 시작된 국제 NGO로서의 정체성 설명.
- **핵심 정보 (리스트):** 
    - 전신(NGI, NKRelief) 통합 정보.
    - 미국 501(c)(3) 비영리 기관 인증.
    - KAICAM 소속 정보.

### 3.3 Core Values Section (비전 및 사명)
- **Vision:** 한반도의 영적 재건과 복음화를 시각적으로 강조된 박스에 배치.
- **Mission:** 기독교 정신 기반의 돌봄과 교육 사명을 6가지 Mission 항목으로 나누어 **3x2 그리드**로 배치.

### 3.4 Ministry Section (4대 사역)
- **구성:** 4개의 가로형 카드.
- **항목:** 동원(Mobilization), 훈련(Training), 네트워킹(Networking), 구호(Relief).
- **디자인:** 각 카드별 아이콘과 설명 포함, 호버 효과 적용.

### 3.5 Founder Section (설립자 케네스 배)
- **구조:** 좌측 사진, 우측 프로필 텍스트.
- **강조 타이틀:** "735일간의 북한 억류, 그리고 그 이후의 사명"
- **프로필:** 학력(오리건 대학, 커버넌트 신학대학원), 안수 정보, YWAM 선교 경력 등.
- **저서:** '잊지 않았다' 관련 언급 및 이미지 플레이스홀더.

### 3.6 Footer / CTA
- **공식 홈페이지 연결:** `www.newkoreafi.org` 링크 버튼.
- **하단 이동:** 메인 페이지(`index.html`) 바로가기.

## 4. 기술적 고려 사항
- **다국어 지원:** `data-i18n-key`를 활용하여 `script.js`의 번역 객체와 연동 가능하도록 설계.
- **스타일 유지:** 기존 `style.css`의 변수(`--primary-color`, `--accent-color` 등)를 재사용하여 디자인 일관성 유지.
