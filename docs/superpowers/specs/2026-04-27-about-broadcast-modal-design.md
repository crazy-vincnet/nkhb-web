# 디자인 문서: 대북방송의 역할 팝업(모달) 전환

- **작성일:** 2026-04-27
- **상태:** 승인됨
- **관련 섹션:** 방송 배경 (`#background`), 신규 모달 (`#article-modal`)

## 1. 개요
별도의 페이지(`about-broadcast.html`)로 이동하는 대신, 메인 페이지 내에서 팝업(모달)을 통해 대북방송의 역할에 대한 상세 내용을 제공하여 사용자 경험의 단절을 최소화합니다.

## 2. 디자인 전략
- **몰입감 있는 읽기:** 모달 내부를 아티클 스타일로 구성(넓은 줄 간격, 적절한 여백)하여 가독성을 확보합니다.
- **편의성:** 스크롤이 가능한 모달 구조를 채택하여 긴 글을 읽기 편하게 합니다.
- **간결함:** 불필요한 페이지 이동을 없애고 사이트의 일관성을 유지합니다.

## 3. 상세 설계

### 3.1 HTML 구조 (`index.html`)
- **버튼 수정:** 기존 링크 버튼을 모달 트리거 버튼으로 변경.
- **모달 추가:** 페이지 하단에 `id="article-modal"`을 가진 새로운 모달 구조 추가.
    - `modal-content` 내부의 `article-content` 영역에 서술형 텍스트 배치.

### 3.2 CSS 스타일링 (`style.css`)
- **모달 확장:** `.modal-content.wide` 클래스를 추가하여 아티클 전용 넓이(800px) 제공.
- **내부 스크롤:** `.modal-body`에 `max-height`와 `overflow-y: auto`를 적용.
- **아티클 스타일 적용:** 모달 내 텍스트에 폰트 크기 17px, 줄 간격 1.8 적용.

### 3.3 기능 구현 (`script.js`)
- **트리거 연결:** '자세히 읽기' 버튼 클릭 시 `#article-modal` 활성화.
- **닫기 기능:** 닫기 버튼 및 모달 외부 클릭 시 비활성화.
- **스크롤 제어:** 모달이 열릴 때 배경(`body`) 스크롤을 방지하여 집중도 향상.

## 4. 정리 작업
- `about-broadcast.html` 파일 삭제.
- `index.html` 내 불필요한 외부 링크 제거.
- `style.css` 내 아티클 전용 페이지 관련 스타일 제거 및 모달 스타일로 통합.
