# 디자인 문서: 후원 섹션 개편 및 결제 링크 연결

- **작성일:** 2026-04-27
- **상태:** 승인됨
- **관련 섹션:** 후원 안내 (`#support-sub`)

## 1. 개요
기존 후원 섹션의 '일시 후원' 항목을 삭제하고, 정기 후원 및 방송 제작 후원 버튼에 직접적인 결제 링크를 연결하여 사용자의 후원 프로세스를 간소화합니다.

## 2. 디자인 전략
- **집중과 선택:** 일시 후원을 제거하여 정기 후원 및 특정 목적(방송 제작) 후원으로 사용자의 관심을 유도합니다.
- **직관적인 연결:** 각 금액 버튼에 대응하는 결제 페이지 링크를 직접 연결하여 이탈률을 낮춥니다.
- **확장된 선택지:** 방송 제작 후원 단위를 세분화하여 다양한 규모의 후원이 가능하도록 합니다.

## 3. 상세 설계

### 3.1 정기 후원 버튼 구성 (6개)
- 3만원: `...price=30000&background`
- 5만원: `...price=50000&background`
- 10만원: `...price=100000&background`
- 20만원: `...price=200000&background` (신규 추가)
- 30만원: `...price=300000&background`
- 50만원: `...price=500000&background`

### 3.2 방송 제작 및 송출 지원 구성 (6개)
- 20만원 (30분): `...oneoff&price=200000`
- 40만원 (1시간): `...oneoff&price=400000`
- 80만원 (2시간): `...oneoff&price=800000` (신규 추가)
- 120만원 (3시간): `...oneoff&price=1200000` (신규 추가)
- 300만원 (한 주간): `...oneoff&price=3000000`
- 1,200만원 (한 달): `...oneoff&price=12000000`

### 3.3 레이아웃 변경 사항
- `index.html`: `support-item` 중 '일시 후원' 영역 전체 삭제.
- `index.html`: 정기 후원 영역의 그리드 버튼을 6개로 업데이트.
- `index.html`: 방송 제작 후원 영역의 그리드 버튼을 6개로 업데이트.
- 모든 링크에 `target="_blank"` 및 `rel="noopener noreferrer"` 적용.

## 4. 기대 효과
- 결제 단계 축소를 통한 후원 전환율 향상.
- 목적이 분명한 후원 옵션 제공으로 사용자 만족도 증대.
