# 디자인 문서: 방송 시간 섹션 다크 테마 전환 및 섹션 분리

- **작성일:** 2026-04-27
- **상태:** 승인됨
- **관련 섹션:** 후원 안내 (`#guide`), 방송 시간 및 주파수 (`#schedule`)

## 1. 개요
현재 인접한 '후원 안내' 섹션과 '방송 시간' 섹션의 시각적 경계가 모호한 문제를 해결하기 위해, 방송 시간 섹션에 다크 테마를 적용하여 명확한 시각적 대비를 제공하고 정보를 강조합니다.

## 2. 디자인 전략
- **강한 대비:** 밝은 배경의 후원 섹션과 대비되는 다크 배경을 방송 시간 섹션에 적용하여 영역 전환을 직관적으로 인지하게 합니다.
- **플로팅 카드:** 어두운 배경 위에 밝은 카드를 배치하여 중요한 정보(시간, 주파수)를 부각시킵니다.
- **전문성 강조:** 다크 테마를 통해 기술적 정보(주파수 등)가 포함된 섹션의 신뢰도와 전문적인 분위기를 연출합니다.

## 3. 상세 설계

### 3.1 섹션 배경 및 타이포그래피 (`.schedule`)
- **배경색:** `#0a192f` (다크 네이비) 또는 기존 `.reach` 섹션과 유사한 딥 다크 톤.
- **제목(h2):** 흰색 (`#ffffff`) 적용.
- **설명(p.description):** 연한 회색 (`rgba(255,255,255,0.7)`) 적용.
- **섹션 태그:** 기존 블루 톤 유지하되 배경 대비 최적화.

### 3.2 카드 디자인 (`.schedule-modern-card`)
- **배경:** 순백색 (`#ffffff`) 또는 아주 연한 그레이 유지.
- **그림자:** `0 20px 40px rgba(0,0,0,0.3)` 등 더 깊은 그림자 적용.
- **텍스트:** 카드 내부의 텍스트는 기존 다크 톤 유지하여 가독성 확보.

### 3.3 하단 안내 문구 (`.notice-item`)
- **배경:** 어두운 배경에 맞는 반투명 배경 또는 짙은 블루 톤으로 변경.
- **텍스트:** 흰색 또는 밝은 톤 적용.

## 4. 기대 효과
- 사용자가 페이지를 스크롤할 때 정보의 성격이 바뀌었음을 즉각적으로 인지.
- 방송 정보(시간, 주파수)에 대한 주목도 향상.
- 웹사이트 전반의 디자인 깊이와 현대적인 감각 강화.
