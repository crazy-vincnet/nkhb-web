# 디자인 문서: 후원 금액 직접 입력 버튼 추가

- **작성일:** 2026-04-27
- **상태:** 승인됨
- **관련 섹션:** 후원 안내 (`#support-sub`)

## 1. 개요
고정된 금액 외에 사용자가 원하는 금액을 자유롭게 입력하여 후원할 수 있는 '직접 입력' 경로를 추가하여 후원 참여의 유연성을 높입니다.

## 2. 디자인 전략
- **높은 가시성:** 정기 후원 금액 리스트 바로 하단에 와이드 버튼 형태로 배치하여 시인성을 확보합니다.
- **명확한 유도:** "직접 입력"이라는 명칭을 통해 고정 금액 외의 선택지가 있음을 명확히 알립니다.
- **안정적인 레이아웃:** 기존 버튼 그리드와 조화를 이루면서도 독립적인 액션으로 인지되도록 스타일링합니다.

## 3. 상세 설계

### 3.1 직접 입력 버튼 명세
- **문구:** 원하는 금액으로 후원하기 (직접 입력) →
- **링크:** `https://secure.donus.org/ngi/pay/step1?dontype=CMP&period=oneoff&price=&background=NKFI`
- **위치:** `index.html` 내 `.support-item`의 `.tier-buttons` 바로 다음에 배치.
- **속성:** `target="_blank"`, `rel="noopener noreferrer"` 적용.

### 3.2 스타일링 포인트
- **너비:** 부모 컨테이너(`support-item`)에 꽉 차는 100% 너비 적용.
- **여백:** 상단 버튼 그리드와 15px 정도의 간격 유지.
- **배경:** 투명 배경에 강조색 테두리를 사용하여 부담스럽지 않은 디자인 유지.

## 4. 기대 효과
- 제시된 금액 외에 소액 또는 고액 후원을 원하는 사용자의 참여 유도.
- 후원 옵션의 다양성 확보를 통한 긍정적인 사용자 경험 제공.
