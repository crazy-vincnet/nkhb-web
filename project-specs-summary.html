1. 비주얼 에디터 고도화 (2026-05)

글로벌 테마 시스템 신규

사이트 전체의 시각적 정체성(색상, 폰트, 간격)을 한 곳에서 제어하는 시스템.

  • CSS Variable 아키텍처를 통한 실시간 스타일 주입
  • 브랜드 컬러, 타이포그래피, 버튼 형상(Border Radius) 통합 관리
  • Visual Editor 내 'THEME' 탭을 통한 실시간 프리뷰 지원
기술: CSS Variables (:root), I18nProvider 확장, postMessage 브릿지

에디터 히스토리 시스템 신규

관리자 편집 작업의 안전성을 위한 실행 취소(Undo) 및 다시 실행(Redo) 기능.

  • Dual-Stack 아키텍처 (Undo/Redo Stack) 기반
  • 50단계의 최대 히스토리 깊이 지원
  • 입력 완료(onBlur) 및 드래그 앤 드롭 완료 시점의 스마트 캡처
  • Cmd+Z / Cmd+Shift+Z 단축키 지원

섹션 순서 변경 (Drag & Drop) 신규

하드코딩된 페이지 구조를 동적으로 렌더링하고 관리자가 순서를 변경할 수 있는 기능.

  • JSON 배열 기반의 레이아웃 데이터 관리 (`page_layout_home`)
  • Component Registry를 통한 동적 컴포넌트 렌더링
  • HTML5 Native Drag & Drop API를 활용한 가벼운 구현

그레이디언트 디자인 & 콘텐츠 센터

개별 요소의 텍스트, 링크뿐만 아니라 세부 스타일(폰트 크기, 색상)까지 관리.

  • `style_props` JSONB 컬럼을 통한 개별 요소 스타일 저장
  • 다국어 엔진(i18n) 업그레이드를 통한 텍스트+스타일 패키징 반환
  • WYSIWYG 방식의 편집 환경 지향

2. UI/UX 및 모바일 최적화 (2026-04)

About 페이지 모바일 최적화

다양한 디바이스 환경에서의 가독성 및 사용성 개선.

  • 반응형 그리드 시스템 적용 (Mobile-First)
  • 이미지 및 텍스트 레이아웃의 유연한 조정
  • 터치 타겟 최적화 및 폰트 크기 스케일링

방송 소개 모달 전환

  • React Portal 기반의 경량 모달 구현
  • 다국어(KO/EN) 콘텐츠의 동적 로딩

다크 테마 스케줄 섹션

가독성을 높이기 위해 스케줄 섹션에 전용 다크 테마 디자인 적용.

  • Tailwind 'dark' 클래스 기반 또는 전용 섹션 클래스 적용
  • 주파수 및 시간 정보의 시각적 강조
